# 07. 힙 정렬 (Heap Sort)

최대 힙의 성질을 이용해 배열을 정렬합니다.

## 핵심 함수 (`main.c`)
| 함수 | 역할 |
|---|---|
| `swap` | 두 원소 교환 |
| `resortHeap` | 한 노드 기준 **아래로 내리기(heapify/sift-down)** — 자식 중 큰 쪽과 비교·교환 |
| `BuildMaxHeap` | 배열 전체를 최대 힙으로 만들기 |
| `BuildMaxHeapAndSort` | 최대 힙을 만든 뒤 정렬 수행·출력 |

## 알고리즘
힙 정렬은 두 단계입니다.

**1단계 — 최대 힙 만들기 (`BuildMaxHeap`)**
마지막 비단말 노드부터 루트까지 거꾸로 `resortHeap`을 호출해
배열 전체가 최대 힙 성질을 만족하게 만듭니다. → **O(n)**

**2단계 — 정렬 (`BuildMaxHeapAndSort`)**
```
반복 (끝에서부터):
  루트(최댓값)와 배열의 마지막 원소를 교환  → 최댓값이 제자리로 확정
  힙 크기를 1 줄이고
  루트에서 resortHeap(아래로 내리기)로 다시 최대 힙 복구
```
최댓값을 하나씩 뒤로 빼내며 정렬 영역을 넓혀갑니다.

## 특징
- 시간 복잡도: **O(n log n)** (최선·평균·최악 모두 동일)
- **제자리 정렬(in-place)** — 추가 배열이 거의 필요 없음 (O(1) 공간)
- 단, 같은 값의 상대 순서가 보장되지 않는 **불안정 정렬**

배열 위에서 힙을 다루는 인덱스 계산(부모 `i/2`, 자식 `2i`, `2i+1`)이
[`../06-heap`](../06-heap)과 이어집니다.
