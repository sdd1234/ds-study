# 00. C 기초 연습

자료구조 본격 진입 전, 주차별로 작성한 C 기초 연습 모음입니다.
파일 이름은 `주제_월-일.c` 형식이며, 한 학기 초반의 학습 흐름을 보여줍니다.

## 주제별 파일

### 재귀 (`recursion_*`)
반복문 버전과 재귀 버전을 함께 구현해 비교합니다.
- **팩토리얼**: `factorial_iter` vs `factorial_rec` — `n! = n × (n-1)!`
- **거듭제곱**: `power_iter` vs `power_rec` — `nʲ = n × nʲ⁻¹`

재귀의 핵심은 **종료 조건(base case)** 과 **자기 호출(recursive case)** 입니다.
거듭제곱처럼 단순 선형 재귀는 반복문으로도 쉽게 바꿀 수 있다는 걸 직접 확인합니다.

### 소수 (`prime_*`)
2부터 n까지 순회하며 소수를 판별/합산합니다. 어떤 수 i가 소수인지는
`2 ~ i-1`(또는 `√i`까지)로 나눠떨어지는지로 판단합니다.

### 다항식 (`polynomial_*`)
- 다항식을 계수 배열로 표현해 출력 (`aₙxⁿ + … + a₁x + a₀`)
- 두 다항식의 곱셈: 각 항끼리 곱해 차수가 같은 자리에 누적

### 스택 (`stack_*`)
배열 기반 스택 ADT. `push`/`pop`/`top`으로 LIFO(후입선출)를 구현하고,
동적 할당(`malloc`)으로 용량을 관리하는 버전까지 발전합니다.
→ 완성형은 [`../03-tree-traversal`](../03-tree-traversal)에서 트리 순회에 활용.

### 연결 리스트 (`linkedlist_*`)
`listNode` 구조체(데이터 + 다음 노드 포인터)로 만드는 단일/원형 연결 리스트.
`insert_first` 등으로 노드를 잇고 끊는 포인터 조작을 연습합니다.
배열과 달리 **삽입·삭제가 O(1)** 이지만 임의 접근은 안 된다는 특성을 익힙니다.

### 큐 (`queue_*`)
FIFO(선입선출) 큐. 연결 리스트로 구현하며 `enqueue`/`dequeue`를 다룹니다.
→ 완성형은 [`../04-level-order-traversal`](../04-level-order-traversal)에서 레벨 순회에 활용.

> 이 폴더의 코드들은 같은 개념을 여러 번 다시 짠 연습본이라 완성도가 제각각입니다.
> 정제된 버전은 위 링크의 상위 단계 폴더에서 볼 수 있습니다.
