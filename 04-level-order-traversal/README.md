# 04. 레벨 순회 + 트리 속성 계산

**큐**를 이용해 이진 트리를 레벨 순서(위→아래, 왼→오)로 순회하고,
순회 과정에서 트리의 여러 속성을 함께 계산합니다.

## 파일 구성
| 파일 | 역할 |
|---|---|
| `LevelTraversal.h` / `.c` | `TreeNode` 정의, 레벨 순회 및 트리 속성 함수들 |
| `Queue.h` / `.c` | 연결 리스트 기반 큐 (`enqueue`/`dequeue`) |
| `main.c` | 트리 생성 후 각 속성 출력 |

## 레벨 순회 알고리즘 (`level_order_x`)
너비 우선 탐색(BFS)과 동일한 원리입니다.

```
큐에 루트를 enqueue
큐가 빌 때까지:
  ptr = dequeue        // 현재 레벨의 노드를 꺼내 방문
  ptr->left 가 있으면 enqueue
  ptr->right 가 있으면 enqueue
```

큐가 FIFO이기 때문에 **먼저 들어온(=위 레벨, 왼쪽) 노드가 먼저 나옵니다.**
그래서 자연히 레벨 순서로 방문하게 됩니다.

## 함께 구현한 트리 속성 함수
| 함수 | 계산 내용 | 방식 |
|---|---|---|
| `GetHeightOfTree` | 트리 높이 | 좌/우 서브트리 높이 중 큰 값 + 1 (재귀) |
| `GetNumberOfNodes` | 전체 노드 수 | 좌 + 우 + 1 (재귀) |
| `GetSumOfNodes` | 노드 값의 합 | 좌 합 + 우 합 + 자기 값 |
| `GetNumberOfLeafNodes` | 잎(leaf) 노드 수 | 좌·우 자식이 모두 없으면 1 |
| `GenerateLinkTree` / `PlaceNode` | 테스트용 트리 생성 | 방향(좌/우) 지정해 노드 연결 |

레벨 순회는 **큐(BFS)** 로, 높이·개수·합 같은 속성은 **재귀(DFS)** 로 푸는 게
자연스럽다는 대비를 한 폴더에서 확인할 수 있습니다.

- 레벨 순회 시간 복잡도: **O(n)**, 공간: **O(w)** (w = 트리의 최대 너비)
